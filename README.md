# 🌌 NASA APOD Explorer

A high-end space-themed gallery that fetches images from NASA’s **Astronomy Picture of the Day (APOD)** API.
Explore the universe daily — or even see what space looked like on your birthday 🚀

---

## ✨ Features

* 📅 **Date Picker**
  Select any date to view historical APOD images.

* 🎂 **Birthday Explorer**
  Discover what the universe looked like on your birthday.

* 🖼️ **Full-Screen Image Modal**
  Click on images to view them in immersive full-screen mode.

* 🔍 **HD Toggle**
  Switch between standard and high-definition images.

* 🌙 **Dark Space Theme**
  Sleek UI with glowing borders and cosmic aesthetics.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/nasa-apod-explorer.git
cd nasa-apod-explorer
```

---

### 2. Get NASA API Key

* Go to: [https://api.nasa.gov/](https://api.nasa.gov/)
* Generate your API key
* Replace it in your project:

```js
const API_KEY = "YOUR_API_KEY_HERE";
```

---

### 3. Run the Project

#### If it's a simple frontend:

```bash
open index.html
```

#### If using React:

```bash
npm install
npm start
```
